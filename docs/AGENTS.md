# AGENTS.md

## Project
Šedý Mentor is a Czech digital brand built as a long-term, respectable, funnel-first system.
It is not a guru brand, not a hype brand, and not a blackhat-operational brand.

Main mantra:
Zkušenost → Dovednost → Možnost

Core meaning:
The goal is not to master everything perfectly.
The goal is to understand systems deeply enough to use them effectively in practice.

---

## Brand Positioning
Šedý Mentor is a strategic system mentor.

The brand should feel:
- intelligent
- calm
- structured
- practical
- premium
- slightly mysterious

The brand should NOT feel:
- scammy
- flashy
- manipulative
- childish
- loud
- overpromising
- “make money fast” oriented

---

## Tone of Voice
Use:
- calm and direct language
- clear structure
- short to medium sentences
- practical wording
- no-bullshit clarity
- grounded confidence

Avoid:
- guru language
- motivational clichés
- fake urgency
- exaggerated promises
- aggressive sales tone
- corporate fluff

Preferred audience reaction:
- “this helps me”
- “this makes me think differently”

Not:
- “this guy is a genius”
- “this feels like an internet guru funnel”

---

## Core System
The project uses a 3-zone architecture:

### Bílá Zóna
Entry / onboarding layer.
Purpose:
- first contact
- current Drop access
- system orientation
- low-friction entry

### Šedá Zóna
Main working layer.
Purpose:
- prompt library
- AI bots
- tools
- Drop archive
- practical digital leverage

### Černá Zóna
Founders / insider layer.
Purpose:
- early access
- internal updates
- expanded archive
- visual prompts
- Project S7
- internal development visibility

---

## Drop Logic
Drop is a recurring content mechanism used for attraction, onboarding, and upgrade motivation.

Rules:
- White Zone sees only the current Drop
- Grey Zone sees all Drops
- Drops must feel useful, practical, and immediately usable
- Drops are part of the system, not random freebies

---

## Project S7
Project S7 is an internal experimental framework available only in the Black Zone.

It is based on archetypes of human motivation inspired by the 7 deadly sins, reframed as:
study of motivation → test in practice → scaling of results

Do not frame S7 as glorification of manipulation, crime, or unethical behavior.
Frame it as study of motivation and experimental system design.

---

## Website Direction
The website should function primarily as a direct funnel, not a blog or corporate site.

Main goal:
move users into the system and toward the right zone

Preferred logic:
- clear hook
- clear value
- simple progression
- low-friction entry
- strong upgrade path

Do not build pages like generic agency websites or content portals.

---

## Visual Direction

### White
entry point
clean
minimal
open

### Grey
main system layer
strategic
balanced
practical

### Black
founders layer
deeper
controlled exclusivity
insider energy

Visual output should feel premium, minimal, and coherent across all three layers.

---

## Hard Rules
Do not:
- create blackhat operational guides
- glorify manipulation
- use fake scarcity
- use loud funnel clichés
- overpromise results
- make the brand feel like an influencer cash-grab

Do:
- keep the system coherent
- prioritize trust, clarity, and conversion
- support a long-term respectable brand
- show depth without chaos
- keep messaging practical and grounded
